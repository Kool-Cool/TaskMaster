Flask Todo App

The Flask Todo App is a simple web application built using Flask and SQLAlchemy. It allows users to manage their tasks by adding, viewing, updating, and deleting them. The app uses an SQLite database to store task information.

Getting Started

  1. Install Python: Make sure you have Python installed on your computer. If not, download and install it from here.
  2. Install Dependencies: Open a terminal or command prompt and install Flask and SQLAlchemy using pip: 
    pip install Flask SQLAlchemy
  3. Run the App: Save the provided code in a Python file (e.g., app.py). In the same directory as your app.py file, run the following command: 
    python app.py
  4. Access the App: Open a web browser and navigate to http://127.0.0.1:5000/ to use the app.

Routes and Functionality

  • / (Homepage):
    • GET: Renders the list of existing tasks on the homepage, sorted by their creation date.
    • POST: Handles form submission to add a new task. Users can enter a task and submit the form to add it to the database.
  • /delete/:
    • GET: Deletes a specific task with the given id from the database. The id is passed as a parameter in the URL.
  • /update/:
    • GET: Renders the page to edit a specific task with the given id. The content of the task can be modified and saved.
    • POST: Handles form submission to update the task content in the database.

Database Model

The app uses a SQLite database to store task information. The database model is defined by the Todo class with the following columns:

  • id (Integer, Primary Key): Unique identifier for each task.
  • content (String, 200 characters, Not Null): Content of the task.
  • date_created (DateTime, Default: Current UTC Time): Date and time when the task was created.

Templates

The Flask Todo App uses two templates:

  1. index.html: Displays the list of tasks and provides a form to add new tasks.
  2. update.html: Allows users to edit the content of a specific task.

Error Handling

The app provides basic error handling for cases when tasks cannot be added, deleted, or updated due to issues with the database.

Conclusion

Congratulations! You now have a simple Todo app built with Flask and SQLAlchemy. Feel free to expand and modify the app to suit your specific requirements or use it as a foundation to learn more about web development with Flask. Happy coding!
